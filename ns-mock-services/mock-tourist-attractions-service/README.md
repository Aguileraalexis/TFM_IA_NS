# mock-tourist-attractions-service

Servicio informativo de atractivos turisticos por ciudad.

## Caracteristicas
- No maneja reservas.
- Usa el mismo catalogo de ciudades que los servicios de hoteles y vuelos.
- Permite consultar atractivos generales o por ciudad.

## Endpoints
- `GET /ciudades`
- `GET /atractivos`
- `GET /ciudades/{ciudadId}/atractivos`
